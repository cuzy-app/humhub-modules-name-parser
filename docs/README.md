# Name Parser

Normalize the format of users' first and last name

Examples:
- James SMITH-GARCIA => James Smith-Garcia
- james smith_garcia => James Smith-Garcia


## Configuration

- Go to `Administration` -> `Modules` -> `Name Parser` -> `Configure`
